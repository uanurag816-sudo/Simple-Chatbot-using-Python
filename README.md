# 🤖 Simple Chatbot using Python

## 📌 Project Overview

This project is a basic text-based chatbot developed in Python. The chatbot responds to a few predefined user inputs using simple conditional statements. It runs in the command line and continues chatting until the user types **"bye"**.

This project is ideal for beginners who want to learn the basics of Python programming, user input handling, loops, and conditional statements.

---

## ✨ Features

* Greets the user.
* Responds to predefined questions.
* Accepts user input continuously.
* Ends the conversation when the user types **"bye"**.
* Handles unknown inputs with a default response.

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Clone or download this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python Chatbot.py
```
